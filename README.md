# California Housing Linear Regression

## Project Overview
This project builds a Linear Regression model using the California Housing dataset to predict house prices.

## Objective
- Load the California Housing dataset
- Perform Exploratory Data Analysis (EDA)
- Train a Linear Regression model
- Evaluate the model using MAE, RMSE, and R² Score
- Save the trained model

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
California Housing Dataset (Scikit-learn)

## Model
Linear Regression

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
